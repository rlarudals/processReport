🖇 Process Report

▷ DB는 MongoDB를 사용

→ MongoDB는 ORM, ODM으로 Graphql과 연결해 주는 기능


✔️ ORM (Object Relational Mapper) : 객체랑 관계의 설정을 의미


✔️ ODM (Object Document Mapper) : NOSQL 관계형 데이터베이스뿐만 아니라 다양한 데이터베이스를 명칭


▷ BackEnd는 NodeJS를 사용

→ NodeJS는 BackEnd와 Graphql을 연결해 주는 기능


▷ FrontEnd는 React를 사용

→ React는 Apollo로 Graphql과 연결해 주는 기능
